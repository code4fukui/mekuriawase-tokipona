# トキポナめくりあわせ

トキポナ(人工言語)の単語を使ったゲームです。単語の表裏を探しあう記憶力ゲームを楽しめます。

## デモ
https://code4fukui.github.io/mekuriawase-tokipona/

## 機能
- トキポナの単語を使っためくりあわせゲーム
- ランダムな配置でゲームをプレイできる
- リトライボタンでゲームを繰り返し遊べる

## データ
ゲームデータには [tok2jpn](https://github.com/code4fukui/tok2jpn) のトキポナ-日本語辞書を使用しています。
